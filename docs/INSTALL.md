# Installation

1.  **Prerequisites:** >=3.10

2.  **Clone the repository:**

    ```bash
    git clone https://github.com/ti014/count_token_llms
    cd count_token_llms
    ```

3.  **Install dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

    The `requirements.txt` file should contain (at least):

    ```
    tiktoken
    ttkthemes
    ```
